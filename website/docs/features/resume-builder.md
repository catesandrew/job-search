# Resume Builder

Build, tailor, and export professional resumes — with an AI that reviews every bullet and proposes improvements you actually control.

## Resume types

The dashboard supports two resume types:

| Type | Purpose |
|---|---|
| **Base Resume** | Your master resume. All positions, all bullets, full skill list. Not linked to any application. |
| **Job Application Optimized** | A tailored copy linked to one specific application. Bullets and skills tuned for that role. |

**Workflow:** Maintain your base resume as your source of truth. When applying for a role, click **Duplicate** to create a copy, link it to the application, and tailor it — without ever touching your base.

## Creating a resume

1. Go to **Resumes → New Resume**
2. Give it a name (e.g. "Base Resume 2025" or "Acme Corp — Staff Engineer")
3. Fill in each section using the inline editor

## The editor

The resume editor has tabbed sections:

### Profile
- Name, title, contact info (email, phone, location, LinkedIn, GitHub, website)
- Professional summary — written by you or generated by AI

### Experience
Each position has:
- Company, title, start/end date, location
- Description (optional)
- **Bullets** — individual achievement statements

Bullets support **rich text** (bold, italic, links). They're stored as HTML and rendered cleanly in the PDF.

### Skills
Organized into named categories (e.g. "Languages", "Frameworks", "Tools"). Drag to reorder.

### Education
Institution, degree, field of study, graduation year.

### Projects
Side projects and open source — name, description, URL, tech stack.

## AI Auto-Improve

The most powerful feature in the builder. One click — AI reviews every visible bullet and proposes specific rewrites.

### How to use it

1. Open a resume (ideally one linked to an application with a job description)
2. Click **Auto-Improve** in the top toolbar
3. A dialog opens with the improvement configuration:
   - **Model** (defaults to active AI provider)
   - **Focus dimensions** (impact, clarity, keyword alignment, seniority signal, brevity, credibility, role fit)
4. Click **Run** — AI analyzes all bullets in a single pass
5. Proposals appear showing **Before** and **After** for each bullet
6. Review each proposal:
   - **Accept** — replaces the bullet with the AI version
   - **Reject** — keeps the original
   - Or **edit** the proposal inline before accepting
7. When satisfied, click **Apply** — only accepted changes are saved

### What the AI evaluates

Each bullet is scored on seven dimensions:

| Dimension | What it measures |
|---|---|
| **Impact** | Does it quantify outcomes? Revenue, time saved, scale? |
| **Clarity** | Is it unambiguous? Does it avoid jargon without explanation? |
| **Keyword Alignment** | Does it reflect terminology from the job description? |
| **Seniority Signal** | Does language match the target level (Staff, Principal, Director)? |
| **Brevity** | Is it concise? Under ~20 words? No filler? |
| **Credibility** | Is it specific? Concrete details > vague claims? |
| **Role Fit** | Does it speak to what this specific role cares about? |

The AI proposes rewrites on a **1:1 basis** — one input bullet, one output bullet. It will not merge bullets, split bullets, or change the number you started with.

### Tips for best results

- Link the resume to an application that has a complete job description — the AI uses the JD for keyword alignment and role fit scoring
- Run Auto-Improve after you've finished a draft, not during initial writing
- Accept improvements selectively — not every AI suggestion will be better than your original
- The AI preserves your facts; it improves the framing, specificity, and language

## PDF Export

Click **Download PDF** to export any resume. PDFs are generated server-side with Puppeteer for pixel-accurate output.

## Templates

Six layout templates available:

| Template | Style |
|---|---|
| Harvard | Classic, single-column, conservative |
| Chicago | Clean, modern two-tone header |
| Bauhaus | Minimal, geometric |
| Oxford | Traditional academic formatting |
| Neue | Contemporary, sans-serif |
| Miller | Compact, high-density |

Switch templates without losing any content — templates are purely presentational.

## Experience Library integration

The **Experience Library** is your master bank of bullets, independent of any resume. When building a tailored resume:

1. Open the resume editor
2. In any Experience position, click **Add from Library**
3. Browse and select bullets from your library
4. They're copied into the resume position — editing them here doesn't affect the library original

This means you write each achievement once (in the library) and reuse it across any resume without rewriting from memory.
